USER INSTRUCTIONS:
This project is a game created using the Greenfoot program. In this game the idea is to control a doctor who has to find the vaccine in the middle of a city full of people infected by a mysterious virus that quickly became a pandemic. Once you get the vaccine, the hardest part is now to heal the infected by applying the vaccine. It is a simple 2D platform game.
